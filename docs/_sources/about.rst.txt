About
-----

This package provides some infrastructure-level modules,
that every application requires that follows the `12-factor principles <https://12factor.net/>`_.

The sub-modules of this package especially focusing on the following principles:

- `III. Config <https://12factor.net/config>`_: Store config in the environment
- `IX. Disposability <https://12factor.net/disposability>`_: Maximize robustness with fast startup and graceful shutdown
- `XI. Logs <https://12factor.net/logs>`_: Treat logs as event streams
